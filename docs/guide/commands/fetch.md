# `uninstall`

Uninstalls Rye again.

For more information see [Uninstalling](/guide/installation/#uninstalling).

## Example

Uninstall Rye:

```
$ rye self uninstall
```

## Arguments

*no arguments*

## Options

* `-y, --yes`: Skip safety check

* `-h, --help`: Print help (see a summary with '-h')
